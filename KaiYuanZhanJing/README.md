# 开元占经及相关的碎碎念
## 碎碎念
这是一本古书，<br>
虽然是占星的，但是并不是每个细节都全无道理 <br> <br>


表面上是占星，实际上是故人对自然现象的观察与总结，比如对
日晷影子长短的观察，实际上是对[米兰科维奇循环](https://zh.wikipedia.org/zh-cn/%E7%B1%B3%E8%98%AD%E7%A7%91%E7%B6%AD%E5%A5%87%E5%BE%AA%E7%92%B0)的一种观察，并且以此对气候变化进行了一定的解释，虽然相关性不代表因果性，然而这不失为一种新思路，来对地球变化进行一种预测与解释。 

整本书基本可以大致分为：对天体的观察，对大气及大气现象的观测，对地质及地质现象的观测，以及物候意义上的观测

换成电子版也有一千多页的一本书呢，我就慢慢写我的注解好了
